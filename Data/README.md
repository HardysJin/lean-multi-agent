# Data Directory

此目录用于存储多时间尺度分层记忆系统的数据。

## 目录结构

- `vector_db/` - ChromaDB向量数据库持久化存储
  - 用于语义搜索和相似度检索
  - 存储决策记忆的embedding向量

- `sql/` - SQLite结构化数据库
  - 用于精确查询和关系型数据存储
  - 存储决策记录、约束条件等

- `cache/` - 临时缓存
  - LLM响应缓存（回测加速）
  - 新闻数据缓存
  - 其他临时数据

## 注意事项

⚠️ 这些目录已在 `.gitignore` 中，数据不会被提交到git仓库。

⚠️ 在生产环境中，建议定期备份 `vector_db/` 和 `sql/` 中的数据。
